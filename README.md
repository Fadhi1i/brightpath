# Web Project Documentation

## Overview
This project is a web application designed to facilitate user authentication and provide different dashboards for various user roles, including parents, teachers, and administrators. 

## File Structure
The project consists of the following files:

- **index.html**: Login page for user authentication.
- **signup.html**: Registration page for new users to create an account.
- **dashboard-parent.html**: Dashboard for parent users, displaying relevant information and features.
- **dashboard-teacher.html**: Dashboard for teacher users, providing access to tools and information specific to teachers.
- **dashboard-admin.html**: Dashboard for admin users, including administrative tools and features for managing the application.
- **assets/css/style.css**: CSS styles for the project, defining the visual appearance of the HTML pages.
- **assets/js/main.js**: JavaScript code for client-side functionality, such as form validation and dynamic content updates.

## Setup Instructions
1. Clone the repository to your local machine.
2. Open the project folder in your preferred code editor.
3. Open `index.html` in a web browser to access the login page.
4. Navigate to `signup.html` to create a new user account.
5. After logging in, users will be redirected to their respective dashboards based on their roles.
